A fan-made web-based Visual Novel bringing together the idol charm of **Idoly Pride** with the timeline-leaping concepts. 

*"No matter the stage or the story, I will there for you."*

## ‚ú® Key Features
* **Web-Based VN Engine:** A custom, buttery-smooth visual novel engine built for the browser. Supports standard VN features like Auto-play, Skip, and Text Log.
* **Cross-Timeline Memory (Save/Load):** Robust state management allowing players to save and load their progress across different branching paths.
* **Glossary System:** Automatic detection and highlighting of lore-specific terms.
* **Dual-Language Hot-Swap:** Fully supports English and Indonesian. Players can switch languages mid-game without restarting.
* **CG Gallery & Audio Control:** A dedicated gallery system with a fullscreen lightbox and real-time audio sliders for BGM and SFX.

## üõ†Ô∏è Built With
* **Frontend:** React.js, TypeScript, and Tailwind CSS v4.
* **State Management:** Zustand (for global game state and persistent storage).
* **Asset Architecture:** JSON scripts hosted on Vercel, with heavy media assets (BGM, CG, Sprites) served via Cloudflare R2 Storage.

## üöÄ Running Locally
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open the provided local link (usually `http://localhost:5173`) in your browser.

# üáÆüá© Versi Bahasa Indonesia

Sebuah proyek *fan-made* Visual Novel berbasis web yang mempertemukan pesona dunia idol dari **Idoly Pride** dengan konsep pelompatan garis waktu. 

*"Di panggung mana pun, dalam cerita apa pun, aku akan ada di sana untukmu."*

## ‚ú® Fitur Utama
* **Web-Based VN Engine:** Mesin visual novel kustom yang mulus di *browser*. Mendukung fitur standar seperti *Auto-play*, *Skip*, dan Riwayat Dialog (*Log*).
* **Cross-Timeline Memory (Save/Load):** Sistem penyimpanan progres yang tangguh untuk kembali ke titik cabang cerita mana pun.
* **Glossary System:** Deteksi otomatis dan sorotan pada kata-kata penting (*lore*) di dalam dialog, lengkap dengan antarmuka kamus.
* **Dual-Language Hot-Swap:** Mendukung Bahasa Indonesia dan Bahasa Inggris. Pemain dapat mengganti bahasa di tengah permainan secara instan.
* **CG Gallery & Audio Control:** Sistem galeri dengan fitur *lightbox* (gambar penuh) dan pengaturan volume *real-time* untuk BGM/SFX.
